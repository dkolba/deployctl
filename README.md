# deployctl

Command line tool for Deno Deploy.

## Install

```shell
deno install --allow-read --allow-write --allow-env --allow-net --allow-run --no-check -r -f https://deno.land/x/deploy/deployctl.ts
```

## Usage

Run a script:

```shell
deployctl run ./examples/hello.js
```

View the help:

```shell
deployctl -h
```
